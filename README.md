# Проект "Карта Детства" (Регион: Находка)

[cite_start]**Назначение:** Единая платформа поиска кружков[cite: 5].
[cite_start]**Входные данные:** Город, дата рождения ребенка, тип активности[cite: 39].

### Как запустить:
1. `npm install`
2. `npm start`

### [cite_start]Тестовый сценарий (п. 3.9)[cite: 48]:
- Выбрать г. Находка.
- Ввести дату рождения для ребенка 10 лет.
- Результат: Отображение кружка "РоботоТех" на ул. Ленинская.
